# Game Store 

- This is built with .NET Blazor.

## Getting Started

1. Cloning the repository. Go in to the repository.

```bash

git clone https://github.com/sssurendra99/GameStore-Blazor.git
cd GameStore-Blazor
```

2. Build the project

```bash
dotnet build
```

3. Run the project.

```bash
dotnet run --project GameStore.Frontend
```
